# Arjunren Von - Personal Portfolio 🚀

Welcome to the repository for my personal developer portfolio. This project serves as a centralized hub for my background, skills, and featured projects as a Software Engineer, Network Engineer, and Ethical Hacker. 

The site is built with an emphasis on high performance, modern UI/UX design, and buttery-smooth animations without the overhead of heavy JavaScript frameworks.

## ✨ Key Features

* **GSAP Image Sequence:** A custom, 60FPS preloaded image scrubber tied directly to the theme toggle, animating smoothly between 64 frames for a highly interactive profile picture.
* **Themed OS Easter Egg:** The environment reacts to the theme. Light mode reveals a bright Windows 11 interface, while Dark mode triggers a sleek Kali Linux Dragon environment.
* **Parallax Project Slider:** Integrates `Swiper.js` to create a cinematic, swipeable featured projects carousel with true 3D background/text parallax effects.
* **Extensive Tech Matrix:** A categorized, scalable grid showcasing skills across Frontend, Backend, DevOps, Cybersecurity, Networking, and IoT hardware, complete with interactive SVG hover states.
* **Fully Responsive:** Pixel-perfect alignment across mobile, tablet, and desktop viewports utilizing Tailwind CSS utility classes.

## 🛠️ Technologies Used

* **Markup & Styling:** HTML5, Tailwind CSS (via CDN)
* **Logic & Interactivity:** Vanilla JavaScript (ES6+)
* **Animations:** [GSAP (GreenSock)](https://gsap.com/)
* **Carousels:** [Swiper.js](https://swiperjs.com/)
* **Icons:** [Heroicons](https://heroicons.com/) & Wikimedia Commons
